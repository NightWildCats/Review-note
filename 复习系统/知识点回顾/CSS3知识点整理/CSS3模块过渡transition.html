CSS3过渡效果
CSS3过渡效果(transition)
(1).transition-property:表示需要使用过渡效果的属性
(2).transition-duration:表示需要过渡需要花费的时间
(3).transition-timing-function:表示需要过渡的运动方式
(4).transition-delay:表示过渡效果开始执行的时间
transition属性:简写属性，用于简写四个过渡属性
  用法：transition:需要过渡的属性 需要过渡的时间 过渡执行的方式(加速，减速，匀速) 开始执行的时间(延迟执行时间)
:target属性
:target属性
用于选定目标元素的ID后，执行设置的样式
  用法：任何元素:target { 属性样式 }
DOM classlist属性
DOM classlist属性
该属性用于在元素中添加，移除及切换 CSS 类
(1).为元素添加多个类:
  用法：document.getElementById("myDIV").classList.add("mystyle","anotherClass", "thirdClass");
(2).为元素移除一个或多个类:
  用法：document.getElementById("myDIV").classList.remove("mystyle","anotherClass", "thirdClass");
(3).为元素切换到不同的类
  用法：document.getElementById("myDIV").classList.toggle("newClassName");
DOM addEventListener()
DOM addEventListener()
addEventListener()：用于向指定元素添加事件句柄
示例：为
用法:
  document.getElementById("myBtn").addEventListener("click", function(){
    document.getElementById("demo").innerHTML = "Hello World";
  });
示例：修改
  document.getElementById("myBtn").addEventListener("click", function(){
    this.style.backgroundColor = "red";
  });
transform缩放
transform缩放
(1).transform-origin:变形的中心点(使用缩放必须要使用的属性)。可以是百分比或者是像素(px)
缩放：
(1).transform:scale(缩放的倍数)——定义 2D 缩放转换。
(1).transform:scale(x，y)——定义 2D 缩放转换。
(2).transform:scale3d(x，y，z)——定义3D缩放转换。
旋转
(1).transform:rotate(角度)——定义 2D 旋转
(2).transform:rotate3d(x,y,z，角度)——定义3D 旋转
倾斜
(1).transform:skew(x倾斜角度,y倾斜角度)，角度的单位是deg
位移
(1).transform:translate(x,y)——定义位移
(1).transform:translateX(位移量-像素)——定义x轴位移
(1).transform:translateY(位移量-像素)——定义y轴位移
  可以应用其做垂直对齐
transform总结：所有的变形都不会导致页面重新布局。还有GPU加速，手机使用会导致发热和费电
